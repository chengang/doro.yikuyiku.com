0.1-alpha
October 8, 2022
时尚
1. 摒弃 32 位 CPU，摒弃 Android 9 及以下版本支持
2. 主架构使用谷歌重构的 Jetpack Compose 架构
3. 各子模块优先考虑性能，选用新技术不考虑对旧设备的兼容
4. UI 界面遵循 Material Design 3 设计语言开发
快速
1. 不同于其它浏览器 Javascript 插件式的链接匹配模块，使用内置于浏览器的 Native 高效率算法
2. Jetpack Compose 带来了 70% 响应速度的提升, UI 响应更及时
3. 地址栏等大部分交互 UI 移动到屏幕下方，更符合移动设备操作习惯
保护隐私
1. 除 android.permission.INTERNET 不申请其它权限
2. 不请求用于追踪的 URL、图片、Javascript
3. 所有日志、记录均使用 Per Device 的 RSA 密钥存储，即便是开发者也无法拿到用户数据
4. 不使用除 Google 官方提供的未经审计的第三方类库
